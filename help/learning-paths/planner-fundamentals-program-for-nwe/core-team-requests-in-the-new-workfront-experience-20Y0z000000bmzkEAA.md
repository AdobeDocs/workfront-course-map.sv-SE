# Begäranden i den nya Workfront-upplevelsen

The *Begäranden i den nya Workfront-upplevelsen* Workfront One Learning Path har migrerats till [Begär köer för användare](https://experienceleague.adobe.com/?recommended=Workfront-U-1-2022.2.request-queues){target="_blank"} på Experience League
